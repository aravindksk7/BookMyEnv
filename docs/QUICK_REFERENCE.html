üìã Quick Actions

Create a Booking

  1. Bookings ‚Üí + New Booking
  2. Select Test Phase & Dates
  3. Add Environment Instances
  4. Submit

Create an Environment

  1. Environments ‚Üí + Add Environment
  2. Fill: Name, Category (NonProd/PreProd/DR)
  3. Create

Add Instance to Environment

  1. Click Environment ‚Üí Instances tab
    • Add Instance
  3. Fill: Name, Status, Capacity
  4. Create

Deploy Application to Instance

  1. Applications ‚Üí View Details
  2. Deployments tab ‚Üí Deploy to Environment
  3. Select Instance, Version, Model
  4. Deploy

Undeploy Application

  1. Applications ‚Üí View Details
  2. Deployments tab
  3. Click Undeploy icon on row
  4. Confirm

Resolve Booking Conflict

  1. Bookings ‚Üí Conflicts tab
  2. Click Resolve icon (gavel)
  3. Select resolution type
  4. Add notes ‚Üí Apply

Bulk Upload Data

  1. Settings ‚Üí Data Management
  2. Go to Bulk Upload
  3. Select entity tab (9 types available):
    • Environments, Instances, Applications
    • Interfaces, Components (App)
    • App-Instance Links, Infrastructure
    • Interface Endpoints, Component Instances
  4. Download template ‚Üí Fill ‚Üí Upload

View Audit Trail ‚≠ê NEW

  1. Sidebar ‚Üí Audit & Compliance
  2. Use filters: Category, Action, Date Range, Actor
  3. Click event row for full details
  4. Export reports using templates

Create a Release

  1. Releases ‚Üí + New Release
  2. Fill: Name, Version, Dates
  3. Select Environments & Apps
  4. Create
